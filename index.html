¿Por qué esto importa?

La etiqueta NFC no necesitó vulnerar el sistema operativo. Solo necesitó convencerte de abrir una dirección.

Al principio, el sitio identificó tu conexión y datos generales del dispositivo. Después, al autorizar la ubicación, recibió coordenadas con precisión en metros.

En una página maliciosa, esos datos podrían almacenarse, relacionarse con una cuenta, acompañarse de una contraseña capturada o utilizarse para personalizar un fraude.

No abras etiquetas NFC desconocidas y no concedas permisos sin revisar primero el dominio y el propósito de la página.

Información que esta página no puede obtener

Una página normal no puede consultar la dirección MAC, el nombre personal del teléfono, la contraseña de Wi-Fi, el nombre de la red Wi-Fi ni el nombre del propietario.

Tampoco puede acceder automáticamente a fotografías, contactos o archivos. Esos recursos requieren selección, permiso adicional o una aplicación con capacidades distintas.

Demostración educativa. La página no posee una base de datos ni envía resultados a un servidor administrado por el creador. Las consultas de IPv4, red y dirección cartográfica utilizan servicios externos.
